Baltimore Blaze Basketball Website

A Professional Basketball Team Website Project

Project Overview

This is a fully responsive basketball team website for the Baltimore Blaze basketball team. The website showcases team information, season schedule, player roster, and official merchandise store. Built with modern web technologies, it features a sleek black and red color scheme that represents the team's identity.

Website Pages

Homepage

Welcome page with team introduction, quick navigation cards, and next game widget

Bio Page

Team history, mission statement, achievements, and core values

Schedule Page

Full season schedule with filter options and game results display

Roster Page

Complete team roster with 15 players and their information

Shop Page

Team merchandise store with product cards and add-to-cart functionality

Key Features

✅ Fully Responsive Design - Works perfectly on mobile, tablet, and desktop devices

✅ Interactive Elements - Schedule filtering, animations, and hover effects

✅ Game Results - Display wins/losses with scores for completed games

✅ Modern UI/UX - Clean design with smooth animations and transitions

✅ Consistent Navigation - Easy navigation between all pages

✅ Team Branding - Custom logo and red/black color scheme throughout

Technologies Used

HTML5 CSS3 JavaScript SVG Responsive Design

The website uses semantic HTML5 structure, CSS3 with variables and modern layouts (Flexbox/Grid), and vanilla JavaScript for interactivity and animations.

Design Highlights

Color Scheme: Primary Red (#DC143C) and Black (#000000) with white accents

Layout: Maximum content width of 1200px, responsive grid layouts, and modern typography

User Experience: Smooth animations, clear navigation, and intuitive interface design

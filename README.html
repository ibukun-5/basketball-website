üöÄ How to Use

Viewing the Website

  1. Open in Browser
    Simply open index.html in any modern web browser. All pages are linked and will work locally.
  2. Navigation
    Use the navigation bar at the top to switch between pages. Click on cards/buttons to navigate to different sections.
  3. Schedule Page
    Use filter buttons to view All Games, Home Games, or Away Games. View game results for completed games (first 5 games).
  4. Shop Page
    Browse team merchandise. Click "Add to Cart" buttons (visual feedback provided).
